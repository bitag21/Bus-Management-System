# Bus-Management-System

## Introduction

This project is a Bus Management System developed using Java. It is designed to manage buses, passengers, drivers, and ticket bookings. The project applies Object-Oriented Programming concepts together with file handling and database connectivity.

## Features

* Add, display, search, update, and delete bus records
* Manage passenger information
* Manage driver information
* Book, search, update, and cancel tickets
* Save and load ticket data using serialization
 * Read and write ticket information using text files
* Store and retrieve passenger records from a MySQL database

## Technologies Used

* Java
* MySQL
* JDBC
* File Handling
* Serialization
* VS Code

## OOP Concepts Implemented

* Encapsulation
* Inheritance
* Abstraction
* Polymorphism
* Method Overloading
* Method Overriding
* Exception Handling

## Project Structure

* model – Contains Bus, Driver, Passenger, Person, and Ticket classes
* service – Contains business logic for bus and ticket management
* database – Handles database connection and operations
* file – Handles file reading, writing, and serialization
* exception – Contains custom exception classes

## Conclusion

This project demonstrates the practical use of Java Object-Oriented Programming concepts, file handling, exception handling, serialization, and database connectivity through a simple Bus Management System.
